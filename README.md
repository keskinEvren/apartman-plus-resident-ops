# 🏡 komşu.site - Apartman ve Site Yaşam Portalı

komşu.site, modern rezidans ve apartman yaşam operasyonlarını kolaylaştırmak için tasarlanmış birinci sınıf, karanlık cam görünümlü (glassmorphism) temalı B2B SaaS sakin portalıdır.

## 🚀 Özellikler

- **Akıllı Kargo Takip**: Güvenlikle entegre çalışan, anlık tek kullanımlık şifre (OTP) üreten akıllı teslimat bildirimleri.
- **Tesis Rezervasyonları**: Havuz, spor salonu ve sosyal tesisleri doluluk oranlarına göre mobil jestlerle (swipe) kolayca ayırtma.
- **Anlık Duyurular**: Yönetimden gelen duyurulardan ve acil durum uyarılarından anında haberdar olma.

## 📂 Mimari Katmanlar ve Statik Dosyalar

Tüm statik varlıklar (örneğin kurumsal `/public/komsu.jpeg` logosu), **Next.js** standartları doğrultusunda platformun `public/` dizininde konumlandırılmıştır.
Sistem yapısı `/config/asanmod-universal/architecture-map.json` üzerinde tanımlı mimari bölge sınırlarına tam uyum sağlamaktadır.
